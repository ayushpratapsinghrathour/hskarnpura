# +2 High School Karnpura, Durgawati

This repository contains the source code of the official website of  
+2 High School Karnpura, Durgawati (Kaimur, Bihar).

## About
The website is built using HTML and CSS with a clean, responsive design
and AI glow animation effects. It provides information about the school,
admission process, achievements, and contact details.

- CSS3
